    Welcome to WordPress-Gear

    Wordpress-Gear is a compendium of useful developer tools for working with WordPress. It supports the following categories

    • Base Tools
    • Debug Tools
    • Skeleton Themes
    • Theme Frameworks
    • PHP Classes

    Wordpress-Gear is meant to be community driven, please feel free to jump in and add/remove any useful information by submitting a pull request via github.

    To submit pull requests fork this project on github then clone it locally

    $ git clone git://github.com/your_username/your_fork.git
$ cd your_fork
# switch branches 
$ git checkout -b gh-pages remotes/origin/gh-pages
# Edit the html files
$ git push

    This site is hosted on github within the default gh-pages branch. To submit a pull request you can follow these instruction https://help.github.com/articles/using-pull-requests

    Why

    The question on what to use comes up far to often so a valuable but somewhat moderated resource seems useful.

    Contributors

    Please refrain from submitting any commercial products and it's best if the links are to online sites that allow for code browsing (like github/bitbucket..etc).

    You can @mention a GitHub username to generate a link to their profile. The resulting <a> element will link to the contributor's GitHub Profile.
